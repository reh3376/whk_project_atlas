# Phase → Linear Project map

Use this file when creating Linear Projects (one per Phase) and Milestones (one per Sprint).

## Recommended Project names

- P00 — Foundation Setup
- P01 — Modular Platform Architecture
- P02 — SDLC & DevEx Standardization
- P03 — Data Pipeline & Curated Datasets
- P04 — MLOps + SME Supervision
- P05 — Reliability / Observability / Security

## Recommended Milestones

Inside each Project, create milestones like:

- Sprint 00.1
- Sprint 00.2
- Sprint 01.1
- Sprint 01.2
- ...

Tip: keep milestone names consistent with folder names under `Phases/`.
